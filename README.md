# DEBUGGERS UTN – Segundo Semestre (JavaScript)

**Esta es la rama **Proyecto-Java**. Contiene todas las actividades realizadas en este proyecto.**

**El mismo cuenta de un sistema de Gestión para empresas con las siguientes caracteristicas:**
1. Sistema de Ventas: Encargado de recopilar datos de Cliente, Producto, Stock, y con la posiibilidad de generar facturas y guardar ventas.
2. Sistema de Clientes: Sirve para poder revisar y modificar informacion de clientes actuales, agregar clientes nuevos y listarlos.
3. Sistema de Proveedores: Ayuda a identificar a cada uno en particular, con información de la empresa, datos de contacto, etc.
4. Sistema de Productos: aporta la funcionalidad de controlar cada producto dentro de la empresa, control de precios, listados de los mismos, y Stock disponible.
5. Sistema de Empleados: Consiste en toda la informacion referida a cada uno de los empleados de la empresa, datos personales, antiguedades, datos de contacto, etc.
